# sites-bsports-pages

## 项目简介

本仓库用于归档和发布一系列独立的 HTML 页面。这些页面是静态资源，不绑定特定域名或网站，主要用于存档、展示或作为独立页面使用。

## 目录结构

```
sites-bsports-pages/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（如 CSS、JS、图片等）
├── index.html      # 仓库入口页面（可选）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件代表一个独立页面，命名清晰便于识别。
- `assets/`：存放页面所需的公共资源，按类型或页面归类。

## 页面归档说明

- 所有页面均为纯静态 HTML，可直接在浏览器中打开使用。
- 页面内容不涉及具体域名、商业推广或诱导行为，仅作为资料归档。
- 归档的页面可能来自不同时期或不同用途，不保证持续更新或兼容性。

## 使用方式

1. 克隆或下载本仓库。
2. 在浏览器中直接打开 `pages/` 下的任意 HTML 文件，或通过 `index.html` 导航访问。
3. 也可将本仓库部署到任意静态托管服务（如 GitHub Pages）进行在线访问。
